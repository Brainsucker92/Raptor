Raptor is a multiplatform chess interface for freechess.org (aka FICS).

Binary Downloads: (Requires Java JRE 6+ 64 bit)

  • Raptor 1.0 Mac OSX (64 bit)
  • Raptor 1.0 Linux (64 bit)
  • Raptor 1.0 Windows (64 bit)
  • Change Log

To install/run in Linux: (Yes i know it needs a package install)

  • Download Raptor 1.0 Linux (64 bit)
  • Unzip in a directory.
  • Open up terminal.
  • cd to dir unzipped.
  • run './raptor'

To install in OS X: (Planning on trying to get this added to the app store)

  • Download Raptor 1.0 Mac OSX (64 bit)
  • Double click dmg.
  • Open finder window.
  • Drag Raptor.app into Applications.
  • Double click on Raptor in Applications (You might have to go into System Preferences -> Security & Privacy to allow Raptor to execute).

To install in Wndows: (Yes i know it needs an installer)

  • Download Raptor 1.0 Windows (64 bit)
  • Unzip in a directory.
  • Double click on Raptor.jar

Release 1.0. Features include:

  • Change Log For a full list.
  • Bundled stockfish 6.
  • Makeover of Chess Analysis tab. Now shows best 5 moves.
  • Makeover of the Bughouse tabs.
  • Timeseal 2
  • Lots of bug fixes. (including the elusive timeseal bug).
  • Removal of features that nobody ever used.
  • Tweaks that I have been meaning to do for the last few years.
  • Inclusion of all chess sets in the expansion pack.
  • Added more chess square backgrounds.
